# Version History

## v1.0.0 (2026-06-12)
Initial public release.

### UI
- Modern themed dialogs, tooltips, context menus, and buttons
- 16 skins, 100+ color themes, 5 quick-save preset slots
- Slider default markers with proximity glow
- Help window with categorized feature guide
- Tabular numerals on all live values

### Hardware Monitoring
- CPU (temp, load, clock), GPU (temp, load, clock, VRAM), RAM, Network, Disk
- CPU temperature via optional PawnIO sensor driver (downloaded on demand)
- Configurable polling interval (250ms - 5s)

### Features
- Game Mode with hotkey snap and per-mode appearance
- Temperature warnings (flash or gradient)
- Remote monitoring over LAN (TCP + handshake key auth)
- Edge and window snapping with blocklist
- Click-through mode with hotkey toggle
- Dark and light mode
- Run at Windows startup
